# Agent 5 — Report Generator

**Name:** `MediSense - Report Generator`

## Description
Combines all analysis, research and recommendations into a beautiful patient-friendly PDF health report with clear sections, color coded results and actionable guidance.

**Step Name:** `Generate Full Health Report`

### Output Evaluation Prompt
```text
Verify all 10 sections are present and complete.
Confirm language is simple and non-technical throughout.
Check emojis are used in test results table.
Verify disclaimer section is at the bottom.
Confirm 7-day diet table has all days filled.
Check action checklist has both this week 
and this month items.
Verify emergency section includes 108 number.
```
